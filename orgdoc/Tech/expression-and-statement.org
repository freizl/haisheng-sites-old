* Expression
  Something which evaluates to a value.
  Example:
  #+begin_src lisp
1 + 2/x
#+end_src

* Statement 
  A line of code which does something.
  Example:
  #+begin_src lisp
a = 1 + a; 
return 10; 
#+end_src
